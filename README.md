# Silant Service

## Описание проекта

Этот проект представляет собой **полнофункциональное веб-приложение** для управления сервисными записями, рекламациями и техническим обслуживанием для различной техники. Система разработана с учетом требований **резиновости, адаптивности и кроссбраузерности**, а также использует **фирменный стиль "Силант"** с его ключевыми цветами (`#163E6C`, `#D20A11`, `#EBE6D6`) и шрифтом `PT Astra Sans`.

**Ключевые возможности:**

*   **Управление техникой:**
    *   Просмотр списка всей техники.
    *   Детальная информация о каждой единице техники.
    *   Добавление новой техники.
*   **Техническое обслуживание (ТО):**
    *   Просмотр истории ТО для конкретной единицы техники.
    *   Создание нового ТО.
*   **Рекламации:**
    *   Oтслеживание рекламаций (заявок на ремонт) по конкретным единицам техники.
    *   Создание новой рекламации.
*   **Поиск по заводскому номеру:**
    *   **Публичный поиск** для гостей и пользователей, позволяющий получить информацию о комплектации и характеристиках техники.
*   **Справочники:**
    *   Просмотр детальной информации по элементам справочников (узлы отказа, способы восстановления и т.д.).

## URL-маршруты

*   **Публичный доступ (без авторизации):**
    *   `GET /public/search/` - **Поиск техники** по заводскому номеру.

*   **Авторизация:**
    *   `GET /accounts/login/` - **Вход в систему**.
    *   `GET /accounts/logout/` - **Выход из системы**.

*   **Доступ для пользователей с соответствующими правами/группами:**
    *   `GET /machines/list` - **Список всей техники**.
    *   `GET /machines/<int:pk>/` - **Детальная информация о технике**.
    *   `GET /machines/<int:machine_pk>/maintenance/` - **Список ТО для техники**.
    *   `GET /machines/<int:machine_pk>/reclamations/` - **Список рекламаций для техники**.
    *   `GET /machines/<int:machine_pk>/maintenance/add` - **Форма добавления ТО**.
    *   `GET /machines/<int:machine_pk>/reclamations/add/` - **Форма добавления рекламации**.
    *   `GET /common/dictionary/item/<int:pk>/` - Детальная информация по элементу справочника.


## Установка и запуск

### Установка зависимостей

1.  Клонируйте репозиторий проекта:
    ```bash
    git clone https://github.com/victoria-sd/Silant-Service.git
    cd silant_service
    ```

2.  Установите все необходимые пакеты из файла `requirements.txt`:
    ```bash
    pip install -r requirements.txt
    ```

### Запуск проекта

1.  **Запуск сервера разработки:**
    ```bash
    python manage.py runserver
    ```
    Сервер будет доступен по адресу `http://127.0.0.1:8000/`.
    

## Доступ к админке

Админка Django доступна по адресу: `http://127.0.0.1:8000/admin/`.
